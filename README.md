# [Collate Status](https://status.collate.ch)

Uptime monitor and status page for Collate, powered by [Upptime](https://upptime.js.org).

Once the workflows have run at least once, this README is automatically regenerated by the **Summary CI** workflow with a live status table for every monitored endpoint.

- Live status page: <https://status.collate.ch>
- Incident history: see the [Issues](https://github.com/Collate-Labs/status/issues) tab
